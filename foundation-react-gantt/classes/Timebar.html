    Type declaration

    • bottom_resolution: string

      The time unit for the bottom timebar.

    • cursorTime: any
    • end: any

      End of the displayed interval, as moment object.

    • leftOffset: number

      The left side offset, this offset is subtracted from the total width of the timeline to obtain the width of the timebar.

    • selectedRanges: any[]
    • setVerticalGridLines: Function

      It's passed by parent. The vertical grid uses the same intervals as the bottom timebar, it is redundant to calculated them again. This callback passes these intervals to parent.

    • start: any

      Start of the displayed interval, as moment object.

    • timeFormats: any

      Formats for each time unit (@see defaultTimebarFormat)

    • top_resolution: string

      The time unit for the top timebar.

    • width: number

      Total width of the timeline (gantt), mandatory field.
