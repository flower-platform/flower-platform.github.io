    ItemRenderer

    Extends:

    * → ItemRenderer

    Item (segment) renderer. All the properties of an item are copied as properties of this component. Hence the component doesn't use the property item.

    All the properties have corresponding getter methods. We "OOP-ize" them, for the case where a subclass wants to "override" a property. Without this pattern, such an override is not doable in React.

    public getBackgroundGradient(): string source

    Create a linear gradient using the base color (calls getColor()) and a color obtained adjusting the brightness of that color using getGradientBrightness(). The default order of the colors is [brighter gradient color, gradient color]; this order can be reversed if getGradientReverseDirection() is true.

    By default, the background of an item uses a linear gradient, this method should be overriden if this behaviour is not wanted.

    Return:

    string

    linear gradient
